# maven-gettext-plugin-debs
Build Debian Package for Maven 2 Gettext Commons

**Build dependencies**

- debhelper (>= 9)
- cdbs
- default-jdk
- maven-debian-helper (>= 1.5)
- libmaven-plugin-tools-java
- libdoxia-sitetools-java
- libmaven-reporting-impl-java
- libmaven2-core-java
- libplexus-utils2-java

**Tested**

- Debian jessie
